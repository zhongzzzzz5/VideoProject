#   教程
jscolor是一个网页的颜色选择器插件，超级方便开发人员安装和使用
一个简单的输入框或者一个简单的按钮就可以获取颜色的插件  非常好用！  
---
+   使用方法
##### 1.只需要要引用jscolor.js
```html
<script src="jscolor.js"></script>
```
#####   2.html
方法1：添加类jscolor
```html
<input class="jscolor" value="ab2567">
```
方法2：添加data-jscolor自定义属性
```html
<input data-jscolor="">
```
